If Win\+Sock, unload the Win\+Sock DLLs; otherwise do nothing. We ignore this in our sample client code but include it in the library for completeness. If you are running on Windows and you are concerned about DLL resource consumption, call this after you are done with all \doxylink{class_socket}{Socket} instances. If you execute this on Windows while some instance of \doxylink{class_socket}{Socket} exists, you are toast. For portability of client code, this is an empty function on non-\/\+Windows platforms so you can always include it.
